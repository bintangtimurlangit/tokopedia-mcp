# Tokopedia MCP documentation

**Tokopedia MCP** is a [Model Context Protocol](https://modelcontextprotocol.io/) server for [Tokopedia](https://www.tokopedia.com): search, product detail, shop catalog, order history, and wishlist — over **stdio**, for Cursor, Claude Code, Claude Desktop, and other MCP hosts.
