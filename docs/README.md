# Tokopedia MCP documentation

**Tokopedia MCP** is a [Model Context Protocol](https://modelcontextprotocol.io/) server for [Tokopedia](https://www.tokopedia.com): search, product detail, shop catalog, order history, and wishlist — over **stdio**, for Cursor, Claude Code, Claude Desktop, and other MCP hosts.

| [Installation](./INSTALLATION.md) | npm / `npx`, global install, clone and build from source |
| [Configuration](./CONFIGURATION.md) | Environment variables, cookies, `mcpServers` JSON, Cursor, Claude Code, Claude Desktop |
| [Development](./DEVELOPMENT.md) | Scripts, TypeScript build, local workflow |
| [Releases](./RELEASES.md) | SemVer, npm tags, git tags, publishing, [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) |
